== Description ==

Agency Hub plugin by Digital Cordex. Connects your WordPress site to your Hub dashboard.

Features:
* File integrity scanner with malware detection
* Real-time activity logging
* Automated backups with signed download links
* Two-factor authentication (TOTP)
* IP blocking with CIDR, country, and user-agent rules
* Full file manager with version history
* HMAC-signed secure communication with Hub

== Installation ==

1. Upload the `agency-hub` folder to `/wp-content/plugins/`
2. Activate the plugin via the Plugins menu
3. Go to Settings > Agency Hub
4. Copy your API Key and Secret
5. Paste them into your Hub dashboard under Site Onboarding
